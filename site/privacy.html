Privacy

Plain and short, because the app collects nothing.

The app collects nothing

CertConvert runs entirely on your own computer. It has no analytics and no telemetry, and it does not send your certificates, keys or any usage information anywhere. There are no accounts and no sign-in. The certificates and keys you open are read in memory and are never uploaded.

What it stores locally

The only file CertConvert creates on its own is a single small settings file that holds one preference — whether to check for updates on launch. Nothing sensitive is kept in it, and you can delete it at any time; the app recreates it with defaults.

PlatformSettings file
macOS~/Library/Application Support/CertConvert/settings.json
Windows%APPDATA%\CertConvert\settings.json

Beyond that, CertConvert writes only the files you explicitly save. It keeps no cache, no logs and no history.

The one network action: checking for updates

The only time the app uses the network is when it checks GitHub for a newer version. This is off by default and otherwise happens only when you choose Check For Updates (or run certconvert update). The check contacts GitHub’s public releases API to compare version numbers and, if you ask it to, download a release; GitHub processes that request under its own privacy policy. No personal data is sent, and this is the only outbound connection the app ever makes.

This update check is present only in the builds distributed from GitHub Releases. Copies obtained from an app store receive updates through the store itself and do not perform their own update check.

This website

By default this website loads no analytics and sets no cookies. If Google Analytics (GA4) is later enabled, it runs under Consent Mode with everything denied until you accept, a cookie banner appears, and you can change your choice at any time via this link. Even then it is used only to understand which pages are viewed — not for advertising, and never to identify you personally. Standard server logs may be kept by the hosting provider for security and reliability.

Contact form

If you message us through the contact form, the name, email address and message you provide are emailed to our support mailbox and used only to read and reply to your enquiry. We don’t add you to any mailing list. The form may be protected by Cloudflare Turnstile, which checks the submission isn’t automated; where enabled, Cloudflare processes limited technical data for this under its own privacy policy and sets no tracking cookies.

Downloads and donations

Downloads are hosted on GitHub Releases; when you download a build, GitHub processes the request under its own privacy policy. Donations are handled by Ko-fi; any payment details are processed by Ko-fi and its payment providers under their own privacy policies — they are never seen or stored by CertConvert.
